# Data-Science-Learning-Journey-Titanic-Dataset
As part of my data science learning, I explored the Titanic dataset using Python, Seaborn, and Matplotlib to practice data visualization and exploratory data analysis (EDA).
Here’s what I worked on:
🔹 Countplot – Passenger distribution by embarkation port and gender
🔹 Swarmplot – Age spread across passenger classes
🔹 Pie chart – Gender distribution
🔹 Heatmap – Feature correlations
🔹 Violin plot – Fare distribution by class and gender
🔹 Scatter plot – Relationship between age, fare, and survival

Key takeaways:

Passenger class and fare had a strong influence on survival
Gender distribution is imbalanced, with more male passengers
Higher-class passengers generally paid more and had better survival chances

✔ Data visualization
✔ Understanding real-world datasets
✔ Extracting insights from data
